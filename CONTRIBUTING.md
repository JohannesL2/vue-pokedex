# Contributing to Vue Pokédex! ⚡️

Thank you for your interest in improving this Pokédex! Whether you want to fix a bug, add a new feature, or improve the styling, your help is highly appreciated.

To keep things smooth for everyone, please follow these simple steps:

## 1. How to Code and Test 💻
This project is built using **Vue 3 (Composition API)**, **TypeScript**, **Pinia** (for state management), and **Vite**.

1. **Fork** and clone the repository to your computer.
2. Create a new branch from `main` (e.g., `git checkout -b feat/my-new-pokemon-feature`).
3. Run `npm install` to install all necessary dependencies.
4. Start the local development server with `npm run dev`.
5. Make your changes and ensure everything works correctly in the browser.

## 2. Submit Your Code 🚀
When you are ready to share your work, open a **Pull Request** (PR) against our `main` branch.

* Keep your PR focused on one specific fix or feature.
* Write a short, clear description of what your code changes or adds.
* Please write clean TypeScript and leverage Pinia stores if you are managing global Pokémon data.

Be kind, have fun, and catch 'em all! 🕵️‍♂️
